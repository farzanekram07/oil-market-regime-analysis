# Oil Market Regime Analysis

A data analytics project that investigates volatility, trends, and market regimes in crude oil prices.

## Objectives

- Analyze historical oil price behavior
- Measure volatility and trend persistence
- Detect market regimes
- Generate actionable market insights

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Plotly
- SciPy
- Scikit-Learn
